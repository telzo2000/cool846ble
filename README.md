# cool846ble


cool846ble is a 40% keyboard like alice layout.
<br>
cool846ble use BLE Micro Pro,run Qmk_firmware.
<br>
Gasket mount using O-ring.
<br>
The case is made with a 3D printer.
<br>
This keyboard has 46 keys, all of which are 1U keys.
<br>
